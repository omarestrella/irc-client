# IRC Client

IRC Client

## Installation

* `git clone` this repository
* `npm install`
* `bower install`
* `grunt download-atom-shell`

## Running

* `ember server --port 9000`
* Run Atom for your system:
  - Mac: `./atombinaries/Atom.app/Contents/MacOS/Atom .`
  - Linux: `./atombinaries/atom .`

## Running Tests

* `ember test`
* `ember test --server`

## Building

* `ember build`
